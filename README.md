# 4166-Final-Project

## API Overview
The **Financial Portfolio Management API** is designed to help users manage their investment activities in one place.  
It provides functionality to:
- Create and organize user portfolios
- Add and track owned stocks
- Maintain a personalized watchlist for stocks of interest

This API will serve as the backend system that supports **users**, **portfolios**, **stocks**, and **watchlists**.
